## 🎯 O Projeto

Sistema completo de automação de ofícios judiciais com IA, transformando **3.5 horas** de trabalho manual em **5 minutos** automatizados.

### **Stack:**
- **Frontend:** Next.js 15 + React 19 + Supabase
- **Backend:** Python 3.12 + GCP + Firebase
- **IA:** OCR + LLM + RAG
- **Deploy:** VPS + Docker
